Modelica Development Tooling Help

Modelica Development Tooling (MDT) Plug-In integrates Open Modelica compiler with Eclipse. MDT, together with Open Modelica compiler provides an environment for working with Modelica projects.

Features

MDT is still in the early development stages but it does provide a simple environment for a modelica programmer. Following features are implemented so far:
  • browsing support for modelica classes
  • syntax color highlighting
  • syntax checking

Background

MDT is created by PELAB at Linköping University as part of a Open Modelica effort. Open Modelica aims at creating a complete development environment for Modelica development.

Requirements

MDT requires at least OpenModelica compiler version 1.3 and Eclipse 3.1. Other versions of Open Modelica and Eclipse may work, but have not been tested.

Happy compiling!
