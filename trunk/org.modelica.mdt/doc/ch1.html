Modelica Development Tooling Help

Modelica Development Tooling (MDT) Plug-In extends Eclipse to provide an environment for working with Modelica projects. These features are provided:

  • Modelica Projects management
  • Modelica Sourcecode Editing support

Background

MDT is created by PELAB at Linköping University as part of a Open Modelica effort. Open Modelica aims at creating a complete development environment for Modelica development.

Requirements

To be able to use this package of fine software you will need a number of other packages of fine software. These are:

  • Open Modelica
  • Cygwin
  • C/C++ Development Tooling Eclipse plug-in

Happy compiling!
